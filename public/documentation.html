The Official Documentation for myFlix Web Application

Purpose

This web application is created for movie enthusiasts who want access to information about different movies, directors, and genres. In essence, it as a movie inventory for users. Users can sign up, update their personal information, and create a list of their favorite movies.

Please follow the formats as indicated in the table below.

Request URL Endpoint HTTP Method to Make Requests Request Body Data Format
Please note: use %20 for spaces and use double quotes 		Response Body Data Format
Return a list of all movies /movies GET None

Simply attach /movies to the end of the URL 		JSON object displaying all available movies:

{ "_id": "5fb4868d2696c3a5f9416efc",
"Title": "Parasite",
"Description": "Greed and class discrimination threaten the newly formed symbiotic relationship between the wealthy Park family and the destitute Kim clan.",
"Featured": true }
Return data about a specific movie by title /movies/:Title GET None

Simply attach the name of the movie in this format:
/movies/The%20Matrix.

If there are no spaces, no need to include %20 i.e. /movies/Parasite 		JSON object displaying the requested movie by title:

{ "_id": "5fb4868d2696c3a5f9416efc",
"Title": "Parasite",
"Description": "Greed and class discrimination threaten the newly formed symbiotic relationship between the wealthy Park family and the destitute Kim clan.",
"Featured": true }
Return data about a genre by name /movies/genre/:Name GET None

Simply attach the name of the genre in this format: /movies/genre/Thriller 		JSON object displaying movies that fall under the requested genre, including information about the genre itself:

{ "_id": "5fb4868d2696c3a5f9416efc",
"Title": "Parasite",
"Description": "Greed and class discrimination threaten the newly formed symbiotic relationship between the wealthy Park family and the destitute Kim clan.",
"Featured": true }
Return data about a director by name /movies/director/:Name GET None

Simply attach the name of the director in this format: /movies/director/Bong%20Joon-ho

If there are no spaces, no need to include %20 i.e. /movies/director/Woody 		JSON object displaying data about the requested director, including movies from the director:

{ "name": "Bong Joon-Ho",
"movie": "Parasite",
"age": "51",
"country": "South Korea" }
Allow new users to register /users POST JSON object

Type user information in this format:

{ "name": "Allan Doe",
"password": "2qpe0!",
"email": "allandoe@gmail.com",
"birthday": "02-16-1995" } 		JSON object containing data about the user's newly written information, including an ID:

{ "id": "91ewh2pz",
"name": "Allan Doe",
"password": "2qpe0!",
"email": "allandoe@gmail.com",
"birthday": "02-16-1995" }
Allow users to update their user profile /users/:username PUT JSON object

Type user information in this format:

{ "name": "Allan Doe",
"password": "4!t01qz",
"email": "allandoeupdatedemail@gmail.com",
"birthday": "02-16-1995" } 		JSON object displaying the user's updated information:

{ "id": "91ewh2pz",
"name": "Allan Doe",
"password": "4!t01qz",
"email": "allandoeupdatedemail@gmail.com",
"birthday": "02-16-1995" }
Allow users to add a movie to their list of favorites /users/:username/movies/:movieID PUT None

Simply attach the movie ID in this format: /users/Allan%20Doe/movies/5fb489032696c3a5f9416eff 		JSON object displaying the user's updated FavoriteMovies list.

{ "FavoriteMovies": [ "5fb489032696c3a5f9416eff" ],
"_id": "5fb6f9678281c837f0b71246",
"username": "testuser",
"password": "testpassword",
"email": "testemail@gmail.com",
"birthday": "1994-02-19T08:00:00.000Z" }
Allow users to remove a movie from their list of favorites /users/:username/movies/delete/:movieID PUT None

Simply attach the movie ID in this format: /users/Allan%20Doe/movies/delete/5fb489032696c3a5f9416eff 		JSON object displaying the user's updated FavoriteMovies list.

{ "FavoriteMovies": [],
"_id": "5fb6f9678281c837f0b71246",
"username": "testuser",
"password": "testpassword",
"email": "testemail@gmail.com",
"birthday": "1994-02-19T08:00:00.000Z" }
Allow existing users to deregister from the registry /users/:username DELETE None

Type user's username in this format:
/users/Allan%20Doe

If there are no spaces, no need to include %20 i.e. /users/Woody 		A message indicating that the user has been successfully deleted from the registry.
