The Official Documentation for myFlix Web Application

Purpose

This web application is created for movie enthusiasts who want access to information about different movies, directors, and genres. In essence, it as a movie inventory for users. Users can sign up, update their personal information, and create a list of their favorite movies.

Please follow the formats as indicated in the table below.

Request URL Endpoint HTTP Method to Make Requests Request Body Data Format
Please note: use %20 for spaces		 Response Body Data Format
Return a list of all movies /movies GET None
Simply attach /movies to the end of the URL		 JSON object displaying all available movies:
{ 'title': 'Parasite', 'year': '2019'},
{ "title": "The Princess Bride", "year": "1987" },
Return data about a specific movie by title /movies/:title GET None
Simply attach the name of the movie in this format: /movies/The%20Matrix.
If there are no spaces, no need to include %20 i.e. /movies/Parasite 		JSON object displaying the requested movie by title:
{ 'title': 'The Matrix', 'year': '1999'}
Return data about a genre by name /movies/genres/:name GET None
Simply attach the name of the genre in this format: /movies/genres/thriller 		JSON object displaying movies that fall under the requested genre:
{ 'title': 'Parasite', 'year': '2019'},
{ 'title': 'Get Out', 'year': '2017' }
Return data about a director by name /movies/directors/:name GET None
Simply attach the name of the director in this format: /movies/directors/Bong%20Joon-ho
If there are no spaces, no need to include %20 i.e. /movies/directors/Woody 		JSON object displaying data about the requested director:
{ 'name': 'Bong Joon-Ho', 'movie': 'Parasite', 'age': '51', 'country': 'South Korea' }
Allow new users to register /users POST JSON object
Type user information in this format: { 'name': 'Allan Doe', 'age': '25', 'email': 'allandoe@gmail.com'} 		JSON object containing data about the user's newly written information, including an ID:
{ 'id': '91ewh2pz', 'name': 'Allan Doe', 'age': '25', 'email': 'allandoe@gmail.com'}
Allow users to update their user profile /users/:username PUT JSON object
Type user information in this format: { 'name': 'Allan Jr. Doe', 'age': '30', 'email': 'allandoe@gmail.com'} 		JSON object displaying the user's updated information:
{ 'name': 'Allan Jr. Doe', 'age': '30', 'email': 'allandoe@gmail.com'}
Allow users to add a movie to their list of favorites /users/:username/movies/:_id PUT None
Simply attach the movie ID in this format: /users/Allan%20Doe/movies/ObjectId("2z93yn") 		A message indicating that the user has successfully added a movie to their favorite list.
Allow users to remove a movie from their list of favorites users/:username/movies/:_id PUT Attach the movie ID in this format: /users/Allan%20Doe/movies/ObjectId("2z93yn")
